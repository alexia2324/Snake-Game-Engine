#  🐍 Snake Game Engine
A modular Python-based implementation of the classic Snake game using the Pygame library.
##Description
This project features a fully functional Snake game with custom assets and modular code structure. It was developed to demonstrate:

# 🚀 How to Play
W/A/S/D: Move the snake.
Goal: Eat the green food to grow and increase your score.
Game Over: Avoid hitting the walls or your own tail.

# 📂 Project Structure
src/: Contains the core game logic (game.py).
assets/: Image files for the background and snake head.
main.py: The entry point for starting the application.
